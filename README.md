# nn_c103_qcotd_swave_only


## About the webpage
The webpage is build with [MkDocs](https://www.mkdocs.org)

Install:
```
pip install -r requirements.txt
```

Run locally
```
mkdocs serve
```

Update: Change the markdown files in `doc-src`.

Push:
```
mkdocs build
git add docs/
git commit -m "new version of page ..."
git push
```
